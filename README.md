# DBI Server Online

Servidor local ligero para instalar archivos NSP en Nintendo Switch usando DBI vía red local.

Permite montar una carpeta con archivos NSP en tu PC y acceder a ella directamente desde la consola sin necesidad de cables, MTP o herramientas adicionales.

---

## ¿Qué hace?

- Convierte tu PC en un servidor HTTP local
- Permite a DBI detectar e instalar NSP vía red (LAN)
- Soporta archivos grandes (incluyendo juegos pesados)
- Compatible con instalación directa desde la Switch

---

## Características

- Selección de carpeta personalizada
- Detección automática de IP local
- Soporte de descargas
- Registro de actividad en tiempo real
- Registro automático
  - Carpeta de NSP
  - Puerto
  - Auto inicio opcional

---

## Requisitos

- PC con Windows
- Nintendo Switch con CFW
- DBI instalado en la consola
- Ambos dispositivos en la misma red WiFi

---

## Uso

1. Ejecuta la aplicación  
2. Selecciona la carpeta donde están tus archivos NSP  
3. Inicia el servidor  
4. Copia la URL generada en DBI "Viewing HTTP/FTP/Github server..."
